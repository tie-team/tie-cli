# tie-cli

> CLI tool for Tie

## Installation

```bash
npm i -g tie-cli
```

## Command

### `tie new`

Init a new Tie project.

### `tie dev`

Runs the app in development mode.

Open http://localhost:5001 to view it in the browser.

### `tie test`

Runs the test watcher in an interactive mode.

### `tie build`

Builds the app for production to the build folder.
