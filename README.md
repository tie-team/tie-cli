# tie-cli

> CLI tool for Tie

## Installation

```bash
npm i -g tie-cli
```

## Command

### `tie create`

创建一个新的 Tie 项目

### `tie dev`

以开发模式启动项目

启动成功后访问： http://localhost:5001

### `tie test`

运行单元测试

### `tie build`

构建可部署的版本到 `build` 文件夹

构建成功后，以生产模式启动项目：`NODE_ENV=production node build/app.js`
